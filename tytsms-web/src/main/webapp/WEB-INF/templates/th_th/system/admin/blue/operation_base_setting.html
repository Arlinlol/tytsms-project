基本设置

  • 基本设置
  • 是否启用积分
  • 积分启用后，会员注册、登录会获取对应积分，商家商品可以使用积分抵扣，商品购买成功后可以获取积分
  • 是否启用积分商城
  • 积分商城启用后，商城前台自动出现积分商城链接，同时可以管理积分商城商品
  • 是否启用预存款
  • 预存款启用后，会员可以给自己帐户充值进行交易
  • 是否启用金币
  • 金币功能启用后，店主可通过平台提供的交易方式进行购买，金币可用来购买广告、直通车等
  • 金币兑换
  • 一元人民币可兑换枚金币
  • 是否启用团购
  • 团购启用后，商城前台自动出现团购链接，同时可以管理团购相关信息
  • *团购套餐起价
  • 金币/30天只有购买了团购套餐的用户才可以发布团购，并且团购只能发布团购套餐时间内

店铺等级

  • 管理
  • 新增
    • #if($!edit)
  • 编辑
    • #end
  • *等级名称
  • 允许发布商品数
  • 0表示没有限制
  • 店铺附件空间
  • 0表示没有限制，店铺附件空间用来存放商品图片等所有店铺附件
  • 等级序号
  • 越小显示越靠前
  • 允许子账户个数
  • 允许商家添加的子账户个数
  • 是否需要审核
  • 需要审核等级，用户提交申请后需管理员确认方可开通
  • 是否限制主营类目
  • 主营类目限制后，所选的经营类目的一级类目只能是同一个一级类目
  • 可选模板套数
  • 0(在店铺等级列表设置)
  • 可用附加功能
  • *收费标准
  • 收费标准，在会员开通或升级店铺时将显示在前台
  • 申请说明
  • 申请说明，在会员开通或升级店铺时将显示在前台
